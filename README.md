# Full Stack Open – Course Solutions (Parts 1–5)

This repository contains my implementations and exercises from the University of Helsinki’s “Full Stack Open” course.

The course focuses on modern web development using:

- React
- Node.js
- Express
- REST APIs
- Testing (Jest / Playwright where applicable)
- State management and frontend architecture

Parts 1–5 have been completed, covering frontend fundamentals, backend API development, database integration, and automated testing.
